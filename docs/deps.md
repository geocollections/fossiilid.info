# Used Dependencies

## 📦Nuxt Modules

### Core

- **Nuxt Image:** Used for optimized image handling (responsive sizes, lazy loading). Primarily used in components displaying media-heavy content.

- **Nuxt SEO:** Handles meta tags, Open Graph, and structured SEO configuration across pages.

- **Nuxt UI:** Main UI component library used throughout the app (buttons, modals, layout primitives).

---

### State Management

- **Pinia:** Global state management (e.g. auth state, shared UI state).

- **pinia-plugin-persistedstate:** Persists selected state (e.g. user session) across page reloads using localStorage.

---

### Utilities

- **VueUse (Nuxt):** Provides composables for common patterns (e.g. local storage, media queries, lifecycle helpers).

---

### Localization

- **Nuxt i18n:** Handles multi-language support, routing, and translations.

---

### Styling / Assets

- **Nuxt Google Fonts:** Automatically loads and optimizes project fonts.

## 📦 Key Dependencies

### Mapping & Geospatial Features

- **Leaflet:** Core map rendering library used across the application.

- **Leaflet plugins (Draw, Geoman, etc.):** Enable drawing, editing, and interacting with map features.

---

### UI & Styling

- **Nuxt UI:** Main component library.

- **TailwindCSS:** Utility-first styling system.

---

### Analytics

- **vue-matomo:** Used for tracking user interactions and analytics.

---

### Utilities

- **dayjs:** Lightweight date handling (used where native Date is insufficient).
